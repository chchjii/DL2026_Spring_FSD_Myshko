# QR Generator Customizer (Test Task)

Веб-приложение для создания кастомизированных QR-кодов с сохранением истории.

## Технологический стек
- **Frontend:** React (Vite), TypeScript, Tailwind CSS, Lucide React.
- **Backend:** Node.js, Express, TypeScript, Prisma ORM.
- **Database:** SQLite (локальный файл).

## Как запустить проект

### 1. Бэкенд
1. Перейдите в папку `backend`: `cd backend`
2. Установите зависимости: `npm install`
3. Настройка базы данных: `npx prisma generate`
4. Запустите сервер: `npm run dev`
   *Сервер запустится на http://localhost:3001*

### 2. Фронтенд
1. Откройте новый терминал и перейдите в папку `frontend`: `cd frontend`
2. Установите зависимости: `npm install`
3. Запустите клиент: `npm run dev`
   *Приложение будет доступно по адресу http://localhost:5173*
